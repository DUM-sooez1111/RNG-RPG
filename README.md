# RNG RPG

브라우저에서 실행되는 2D RPG 프로토타입입니다.

## 플레이

GitHub Pages가 배포된 뒤에는 다음 주소에서 누구나 플레이할 수 있습니다.

https://dum-sooez1111.github.io/RNG-RPG/

처음 실행하면 컴퓨터 또는 모바일 조작 방식을 선택할 수 있습니다.

## 조작법

- 이동: 방향키 또는 WASD
- 공격: Space
- 주사위: F
- 동료 스킬: C
- 상호작용: E
- 인벤토리: I
- 스킬트리: K
- 상점: 집 앞에서 Q

모바일에서는 화면의 터치 조작 버튼을 사용합니다.
