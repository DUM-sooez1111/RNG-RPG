# RNG RPG

브라우저에서 실행되는 3D 스타일 RNG RPG 프로토타입입니다.

## 플레이

GitHub Pages가 배포된 뒤에는 다음 주소에서 누구나 플레이할 수 있습니다.

https://dum-sooez1111.github.io/RNG-RPG/

처음 실행하면 컴퓨터 또는 모바일 조작 방식을 선택할 수 있습니다.

## 조작법

- 이동: 방향키 또는 WASD
- 공격: Space
- 주사위: F
- 동료 스킬: C
- 자동 사냥: Z (던전에서 사용)
- 자동 주사위: G 또는 화면의 `자동 주사위` 버튼
- 상호작용: E
- 인벤토리: I
- 스킬트리: K
- 상점: 집 앞에서 Q
- 카메라 180° 회전: 게임 화면 우클릭

인벤토리의 `최고 장착` 버튼으로 보유 장비와 동료 중 가장 강한 구성을 한 번에 장착할 수 있습니다.

## 등급

전설 이후에도 합성과 매우 희귀한 뽑기를 통해 `신화 → 고대 → 신성 → 초월 → 절대` 등급까지 성장할 수 있습니다.

모바일에서는 화면의 터치 조작 버튼을 사용합니다.
